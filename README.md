# Tensorflow 的安裝
我使用 Debian Linux 9.3，相關的環境如下：
1. python v3.5
2. pip v1.18.0
3. venv

Before I start to install the Tensorflow package into my system, I create a
virtual environment as following commands first:
\# python -m venv tensorflow-study
